# Skill 分享 PPT

这是一份用 HTML PPT Skill 生成的网页演示稿。

## 如何查看

直接打开 `index.html` 即可。推荐使用 Chrome、Edge 或 Safari 的最新版浏览器。

## 目录结构

- `index.html`：完整 PPT 页面
- `images/`：页面截图、封面、SVG 视觉和可替换图片素材
- `videos/`：页面视频素材和占位视频

## 素材替换约定

后续替换素材时，优先保持同名文件覆盖：

- 主视频：替换 `videos/{页码}/主视频*.mp4`
- 视频封面：替换 `images/{页码}/主视频*封面.png`
- 第 41 页旧版对比图：替换 `images/41/旧图01.png` 到 `旧图06.png`

当前发布版中的长视频已经裁剪到约 60 秒，避免超过 GitHub 单文件大小限制。正式视频建议也控制在 1 分钟以内。
